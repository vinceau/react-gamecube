# React Gamecube

## Development

### Build

To build the component library, run:

```sh
yarn run build
```

To start the storybook server, run:

```sh
yarn run start
```

### Test

To run the tests:

```
yarn run test
```

### Export Storybook

To export the storybook as static files:

```
yarn run storybook
```

You can then serve the files under `storybook-static` for demonstration.
